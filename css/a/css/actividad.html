1. Flexbox (Caja Flexible)

Flexbox es un modelo de layout unidimensional para diseñar estructuras flexibles.

Propiedades principales: display: flex, flex-direction, justify-content, align-items, flex-wrap

/* Contenedor Flex */ .flex-container { display: flex; flex-wrap: wrap; justify-content: space-between; gap: 15px; } /* Elementos Flex */ .flex-item { flex: 1 1 200px; /* grow, shrink, basis */ background-color: #1a2a6c; color: white; padding: 20px; border-radius: 8px; }

2. CSS Grid

Grid es un modelo de layout bidimensional para diseñar estructuras en filas y columnas.

Propiedades principales: display: grid, grid-template-columns, grid-template-rows, gap, grid-column, grid-row

3. Float (Flotar elementos)

Float permite que un elemento se coloque a la izquierda o derecha, y el texto fluya alrededor.

Propiedades: float: left, float: right, clear: both
Elemento flotante a la izquierda
Elemento flotante a la derecha

Este texto fluye alrededor de los elementos flotantes. Float es una técnica antigua pero aún útil para ciertos casos. Con la llegada de Flexbox y Grid, el uso de float para layouts ha disminuido, pero sigue siendo útil para imágenes dentro de texto o efectos específicos.

Es importante recordar que después de usar elementos float, debemos limpiar el flujo con clear: both o overflow: auto en el contenedor padre para evitar problemas de layout.

/* Elementos flotantes */ .float-left { float: left; width: 150px; margin-right: 15px; } .float-right { float: right; width: 150px; margin-left: 15px; } /* Limpiar floats */ .float-section { overflow: auto; /* Esto limpia los floats */ }

4. Tablas en HTML

Las tablas se utilizan para mostrar datos tabulares. Usa las etiquetas <table>, <tr>, <th> y <td>.

5. Formularios en HTML

Los formularios permiten capturar datos del usuario. Usa las etiquetas <form>, <input>, <label>, <select>, <textarea>, etc.

<form action="#" method="post"> <div class="form-group"> <label for="nombre">Nombre completo:</label> <input type="text" id="nombre" name="nombre" required> </div> <div class="form-group"> <label for="email">Correo electrónico:</label> <input type="email" id="email" name="email" required> </div> <div class="form-group"> <label for="pais">País:</label> <select id="pais" name="pais"> <option value="">Selecciona un país</option> <option value="es">España</option> </select> </div> <!-- Más elementos del formulario --> <button type="submit">Enviar Formulario</button> </form>

6. Conceptos Básicos de HTML y CSS

Estructura básica de HTML

<!DOCTYPE html> <html lang="es"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título de la página</title> <link rel="stylesheet" href="estilos.css"> </head> <body> <h1>Mi página web</h1> <p>Contenido de la página</p> </body> </html>

Selectores CSS comunes

  • selector de elemento: p { color: blue; }
  • selector de clase: .clase { font-size: 16px; }
  • selector de ID: #id { background-color: yellow; }
  • selector descendente: div p { margin: 10px; }
  • selector de atributo: input[type="text"] { border: 1px solid #ccc; }

Box Model (Modelo de caja)

Todo elemento en HTML es una caja con: contenido, padding, border y margin.

Contenido: El contenido real (texto, imagen, etc.)

Padding: Espacio interno entre el contenido y el borde

Border: Línea que rodea el padding y contenido

Margin: Espacio externo entre este elemento y otros
